# Персональний Telegram-асистент

Що вміє ця версія:
- ➕ Створити задачу
- 📋 Список задач
- ✅ Виконати сьогодні
- 📅 Сьогодні
- 💰 Бюджет
- 🛒 Купити продукти
- 📝 Нотатка
- 📦 Що купити
- 🔎 Знайти
- 📍 Місця поруч
- 📊 Звіт за тиждень
- голосові повідомлення через Gemini
- аналіз фото
- запис задач у Google Sheets
- додавання задач із датою/часом у Google Calendar

## Листи в Google Sheets
Бот працює з окремими листами і не чіпає старий `Applications`:
- `Задачі`
- `Бюджет`
- `Покупки`
- `Журнал`
- `Нотатки`

## Як працює календар
- якщо в задачі є дата і/або час, бот створює подію в Google Calendar
- кнопка `📅 Сьогодні` показує і задачі, і події календаря на сьогодні
- кнопка `📊 Звіт за тиждень` показує коротку статистику по задачах, бюджету і покупках

## Що має бути в Cloudflare
Обов'язково:
- `BOT_TOKEN`
- `GEMINI_API_KEY`
- `GOOGLE_SHEET_ID`
- `GOOGLE_SERVICE_ACCOUNT_JSON` (secret)
- `CALENDAR_ID`
- `TIMEZONE`
- `SETUP_KEY`
- `TELEGRAM_WEBHOOK_SECRET`
- `STATE_KV`

## Після заміни файлів
1. Задеплой Worker.
2. Відкрий `/setup?key=ТВОЄ_ЗНАЧЕННЯ_SETUP_KEY`
3. Напиши боту `/start`
